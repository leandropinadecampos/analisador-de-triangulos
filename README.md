# Classificador de Triângulos

Programa em Python que verifica se três valores podem formar um triângulo e classifica como:

- Equilátero
- Isósceles
- Escaleno

## Como usar

Execute o arquivo Python e insira os valores solicitados.

## Conceitos utilizados

- Estruturas condicionais (if/elif/else)
- Funções
- Entrada de dados (input)
